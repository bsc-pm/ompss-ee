Heat diffusion (Jacobi solver)
---------------

.. highlight:: c

This codes performs a ... hybrid MPI/OpenMP implementation.

.. note::
  You can dowload this code visiting the url http://pm.bsc.es *OmpSs Examples and Exercises*'s
